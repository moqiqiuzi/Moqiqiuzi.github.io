vercel的基础使用和nodejs的卸载

发表于2021-09-02|更新于2021-09-02|Hexo
|阅读量:

最近看到vercel，就想看一下怎么用，但是之前不知道什么原因我使用npm一直会出现报错，很久都没有解决捞出来解决一下

一、解决npm版本问题

发现是因为之前版本更新的操作错误，然后多下了两个

1、卸载方式：

​ 步骤一：删除nodejs目录下的所有文件

​ 步骤二：删除.npmrc文件，一般在用户目录下

2、安装：

然后正常安装，用-v检测是否安装成功，设置淘宝镜像：

npm config set registry https://registry.npm.taobao.org

二、简单使用vercel

1、使用npm安装vercel

npm i -g vercel

2、登录到vercel

使用github注册账号，创建库

3、连接本地文件目录

cd到本地网站的目录，第一次连接需要输入邮箱

4、发布网站

vercel –prod

选择合适的库上传本地的网站

以下是我刚发布做测试的，链接不会实时更新，内容和本站基本一样
